Introduction to computing

This beginner-level course provides students with foundational knowledge of computer systems, software applications, and basic programming concepts. Through hands-on activities and projects, students will explore the principles of hardware, software, operating systems, and data representation. The course also introduces logical thinking and problem-solving using simple code structures in languages such as Python or Scratch.

Web development fundamentals

Designed for learners with some basic computing experience, this course teaches the essential skills needed to build and maintain websites. Topics include HTML, CSS, JavaScript, and responsive design principles. Students will develop interactive web pages and learn how to publish and manage them using modern tools and frameworks, emphasizing user experience and accessibility.

Advanced programming and algorithms

This advanced-level course focuses on developing efficient and scalable software solutions using object-oriented programming and algorithmic thinking. Students will explore data structures such as lists, stacks, queues, trees, and graphs while implementing algorithms for sorting, searching, and optimization. Emphasis is placed on writing clean, maintainable code and understanding time and space complexity.
